# 👩🏽‍💻 Meu Portfólio Pessoal

Um portfólio moderno e responsivo desenvolvido para apresentar minha trajetória, projetos e estudos de caso na área de Engenharia de Software e Desenvolvimento Web.

🔗 **[Acessar o Portfólio Online](https://marcelly-ramos.github.io/marcelly-portfolio/)**

## 💡 Sobre o Projeto
Este projeto foi construído do zero, focado em entregar uma interface limpa, navegação fluida e uma excelente experiência do usuário. A arquitetura visual utiliza conceitos modernos de UI Design, como a estrutura em **Bento Box**, para destacar as informações mais importantes de forma assimétrica e organizada.

Além da apresentação pessoal, o portfólio conta com páginas detalhadas de Estudos de Caso (como o Case do Sistema de Digitalização), demonstrando não apenas o resultado visual, mas minha capacidade de mapeamento de requisitos, fluxogramas e visão sistêmica.

## 🛠 Tecnologias Utilizadas
- **HTML5:** Estrutura semântica e boas práticas.
- **CSS3:** Estilização com variáveis globais (Custom Properties), Flexbox, CSS Grid e media queries para total responsividade.
- **JavaScript (Vanilla):** Lógica limpa e sem frameworks para interatividade, navegação e funcionamento de carrosséis.
- **Performance:** Imagens otimizadas (TinyPNG) e estrutura de arquivos web-safe para deploy rápido.
